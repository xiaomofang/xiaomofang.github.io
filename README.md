## 更新博客

修改文件后，在项目目录执行：

```powershell
cd E:\AAA-LLM-CODE\tech-blog-github-pages
git add .
git commit -m "更新内容"
git push
```

推送后刷新网页即可看到更新。

---

## 发新文章

1. 复制 `posts/post-llm-benchmark.html` 为新文件
2. 修改标题和正文
3. 在 `index.html` 添加一篇文章卡片
4. 按上面步骤 `git add` → `commit` → `push`
